Seleção de Variáveis (Parte 2)

Gustavo Santos
Jan 9 · 3 min read

Entenda como utilizar o SelectKBest para escolher as melhores variáveis para o seu modelo de Machine Learning.

A Parte Prática

Na semana passada, escrevi o artigo Estatística para Seleção de Atributos, o qual recomendo a leitura para entender os conceitos teóricos, o que está por trás dessa ferramenta de seleção de variáveis do sklearn. Entretanto, devo dizer que, apesar de ser a segunda parte de um conteúdo, ambos os artigos podem ser lidos de forma independente, servindo ao propósito de te ajudar com essa tarefa.

Métodos de Seleção de Atributos

Selecionar os melhores atributos significa remover aqueles que não melhoram a performance ou apenas adicionam ruído ao modelo.

Existem dois métodos principais que podem ser usados para completar essa tarefa:

  • Método wrapper (embrulho, em tradução livre): são criados vários modelos com diversas combinações de variáveis e escolhe-se aquelas que resultaram na melhor performance para o modelo, de acordo com métricas de avaliação. Uma ferramenta que ilustra esse conceito é a RFE.
  • Método filter (filtro, trad. livre): este é o método em que encaixamos o SelectKBest, onde usamos métodos estatísticos — como chi-quadrado, correlação, ANOVA — para medir a relação entre as variáveis preditoras e a variável target e, com os resultados, podemos filtrar apenas os atributos com melhores valores.

Escolhendo o Filtro Estatístico Ideal

Já que vamos trabalhar com o método de filtro, precisamos saber qual é o melhor a ser usado em cada situação. Esses filtros são baseados em algum tipo de correlação entre as variáveis, mas dependendo de qual tipo (numérico ou categórico) dessas variáveis estamos trabalhando, devemos escolher uma estatística diferente.

Deste modo, esta seção vai tratar de resumir qual é o melhor filtro para escolher em cada um dos casos.

  • Entrada numérica; Saída numérica (modelo de Regressão): Correlação de Pearson para relações lineares. Correlação de Spearman para correlações não lineares.
  • Entrada numérica; Saída categórica (modelo de Classificação): ANOVA
  • Entrada categórica; Saída categórica (modelo de Classificação): Chi-quadrado

Na prática, o filtro estatístico será um hiperparâmetro da função SelectKBest.

Na Prática

Vamos à prática.

A seleção de atributos em Python geralmente é feita após a exploração inicial dos dados, da limpeza e normalização do seu dataset.

E, claro, como tudo em Ciência de Dados, não é uma verdade absoluta. Lembre-se: a ferramenta apresenta a você — Cientista de Dados — aquilo que numericamente retornou melhor performance. Cabe a você, usando de sua experiência, entendimento do negócio do seu cliente e do problema a ser resolvido, avaliar quais atributos farão parte da lista final para seu modelo preditivo.

[In] Numérico ; [Out] Numérico:

SelectKBest(score_func=f_regression, k=numero de atributos desejados)

SelectKBest(score_func=mutual_info_regression, k=num atr. desejados)

[In] Numérico ; [Out] Categórico:

SelectKBest(score_func=f_classif, k=numero de atributos desejados)

[In] Categórico ; [Out] Categórico:

SelectKBest(score_func=chi2, k=numero de atributos desejados)

SelectKBest(score_func=mutual_info_classif, k=num atr. desejados)

Conclusão

Podemos usar o método de filtro baseado em estatística para a seleção das melhores variáveis. Basta, para isso, sabermos como eles funcionam e quais são os melhores para cada situação.
